Opérations possibles:

Selon les types des variables on peut réaliser différentes opérations:

Pour un integer(entier) ces opérations sont : DIV, MOD,*,+,-;

Pour un real(réel) ces opérations sont : /,*,+,-;

Pour une string(chaine de caractères)ces opérations sont: EXTRACT,+,nom_de_la_chaine[i];

Pour un boolean(booléen)ces opérations sont: <,>,=<,>=,<>,=;

Variables:

Les variables servent a stocker des valeurs. Elles possèdent un type qui peut être :

  1. integer (un entier)
  2. real (un réel)
  3. char (un caractère)
  4. string (une chaine de caractères)
  5. boolean (un booléen)
  6. array (un tableau)

Les opérations élémentaires possibles sur les variables sont:

  1. déclaration :

    2. nom de la variable ->nom_de_la_variable:TYPE; <- type de la variable le type de la variable est toujours précédé du signe : et suivi du signe ;
    Expemple: note:REAL;
    ATTENTION: Un tableau se déclare différament voir la partie tableau.
    On peut aussi déclarer plusieurs variables de même type sur la même ligne.
    Expemple: mot1,mot2,mot3:STRING;

  2. affectation :

    3. On peut affecter à une variable :
    1-une valeur : nom de la variable-> nom_de_la_variable:=valeur;ATTENTION la valeur doit être du même type que la variable. Exemple: note:=20;
    2-une variable: nom de la variable à modifier-> nom_de_la_variable:=nom_de_la_variable_2;<- nom de la variable affectée ATTENTION la variable affectée doit être du même type que la variable à modifier. Exemple: mot1:=mot2; ici les deux variables sont des string (chaines de caractères).
    3-une expression: nom de la variable -> nom_de_la_variable:=expression;ATTENTION le résultat de l'expression doit être du même type que celui de la variable. Exemple: note:=15+5;

  3. utilisation :

    4. Pour utiliser une variable il suffit de l'avoir déclaré puis de l'appeler en écrivant son nom. Une variable peut être utilisée dans une expression, un tableau, une procedure ou une fonction.

  4. ré-affectation :

    5. On peut également réaffecter une variable par exemple:note:=note+1;ATTENTION le résultat de l'expression doit être du même type que la variable et les opérations doivent être compatible avec le type de la variable.

Constantes:

Les constantes sont des variables qui ne pourront être changées dans le programme. Elles sont déclarées au début du programme comme suit:

nom_de_la_constante=valeur;

Fonctions pascal:

Il existe différentes fonctions de base dans freepascal dans la bibliohèque crt:

write sert à afficher un/des texte/s et/ou une ou des variables.
ex: write('texte1',variable1,'texte2',variable2);

writeln sert à la même chose que write mais rajoute un retour à la ligne.
ex: writeln('texte1',variable1,'texte2',variable2);

readln sert à sauvegarder une valeur entrée par l'utilisateur dans une variable.
ex: readln(nom_de_la_variable);

slrscr sert à éffacer l'écran.
ex: clscr;

Boucles et conditions:

Il existe différentes boucles :

WHILE condition DO

begin

instructions

end;


REPEAT

instructions

UNTIL condition;


FOR variable:=valeurinitiale TO valeurfinale DO

begin

instructions

end;


Et plusieurs conditions:

IF condition THEN

begin

instructions

end

ELSE

begin

instructions

end;


CASE nom_de_la_variable OF

begin

cas1:instructions
cas2:instructions...
...casn:intructions

end;

Procédures et fonctions:

Dans freepascal il est aussi possible de créer des procédures et des fonctions.
Les procedures exécutent des instructions et ne renvoient rien dans le programme et les fonctions exécutent des instructions et renvoient une valeur de sortie.
Les procédures se déclarent comme suit :

procedure nom_de_la_procedure(paramètre:type_du_paramètre;var paramètre(sert à modifier la variable dans la fonction/procedure et enregistrer les changements):type_du_paramètre);

VAR(variables locales de la fonction)

BEGIN

instructions

END;

Une fonction se déclare comme une procedure, il faut juste rajouter le type de la variable de sortie précédé du signe ':' après la déclaration des paramètres, et remplacer procedure par function.

Exemple:
function calcul(nombre,nombre2:integer):integer;

VAR

res:integer;

BEGIN

res:=nombre+nombre2;
calcul:=res;

END;

Pour appeler une procédure ou une fonction il suffit d'écrire nom_de_la_procedure_ou_fonction(paramètre1,paramètre2...);

Tableaux et types:

Les tableaux se déclarent différament des autres variables. Un tableau se déclare comme suit :
nom_du_tableau:array[1..longueur_du_tableau...autant qu'il y a de dimensions] of type_des_variables_à_l'intérieur_du_tableau;

Pour créer un nouveau type il suffit d'entrer les commandes suivantes:

TYPE

nom_du_type=type

exemple:
TYPE

Tableau2dim=array[1..TAILLE,1..TAILLE] of integer;
